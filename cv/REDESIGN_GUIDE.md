## 🎨 设计特点

### 1. **现代简洁风格**
- 采用深蓝色（#2c3e50）作为主色调
- 浅灰色（#f8f9fa）作为背景
- 蓝色强调色（#3498db）用于交互元素
- 响应式设计，完美适配各种屏幕尺寸

### 2. **视觉层次结构**
- 清晰的导航菜单（固定顶部）
- Hero 区域突出个人信息和主要成就
- 使用卡片（Card）设计组织内容
- 统计数据框展示主要成就（70+论文、23+专利等）

### 3. **内容组织方式**
```

  ├─ 社交媒体链接

  ├─ 搜索功能
  ├─ 期刊论文列表
  └─ 会议论文列表

发明专利 (Patents) - 包含过滤功能
  ├─ 授权状态筛选（已授权, 实审, 申请）
  ├─ 专利类型筛选（中国, PCT）
  └─ 专利列表

大数据系统与演示平台 (Systems)
  ├─ 国家基础学科公共数据中心
  ├─ 学部工作局科技智库
  ├─ 国际合作数据平台
  ├─ 科技合作管理平台
  └─ NSFC 知识图谱平台

  ├─ 近期主持项目
  └─ 历史项目（可展开/收起）

其他信息 (Others)
  ├─ 参与标准
  ├─ 教学活动
  ├─ 指导学生
  ├─ 社会兼职
  └─ 图书章节
```

## 🔍 核心功能

### 1. **论文过滤系统**
支持多维度组合过滤：
- **研究方向**：AI for Science, 科技大数据知识图谱, 数据集与基准
- **论文类型**：期刊论文, 会议论文  
- **时间跨度**：2024-2026, 2020-2023, 2010-2019
- **关键词搜索**：实时搜索标题和作者

### 2. **专利分类系统**
支持二维过滤：
- **授权状态**：已授权, 实审中, 申请中
- **专利类型**：中国发明专利, PCT 专利

### 3. **展开/收起功能**
- 代表性论文的显示隐藏
- 历史专利的显示隐藏
- 历史项目的显示隐藏
- 更多学生信息的显示隐藏

### 4. **搜索功能**
- 全文搜索论文（标题+作者）
- 实时过滤结果
- 支持中英文搜索

## 🚀 使用方法

### 基础访问
1. 在浏览器中打开 `index_zh_new.html`
2. 或用 HTTP 服务器访问：`http://localhost:8000/cv/index_zh_new.html`

### 过滤功能使用
1. **按研究方向过滤** - 点击相应的按钮：全部、AI for Science、知识图谱、数据集
2. **按论文类型过滤** - 选择期刊论文或会议论文
3. **按年份过滤** - 选择时间范围
4. **搜索论文** - 在搜索框中输入关键词

### 专利过滤
1. **按状态过滤** - 选择已授权/实审/申请
2. **按类型过滤** - 选择中国专利或PCT

## 💾 数据更新方法

所有数据存储在 `modern-scripts.js` 中，格式为：

### 添加新论文
编辑 `modern-scripts.js` 中的 `papersData` 对象：
```javascript
papersData.journal.push({
    id: 'J2026-XXX',
    title: "新论文标题",
    authors: "作者列表",
    journal: "期刊名",
    year: 2026,
    sort_year: 2026,
    direction: "ai-for-science",
    type: "journal",
    meta: "SCI IF:XX, ...",
    url: "链接"
});
```

### 添加新专利
编辑 `modern-scripts.js` 中的 `patentsData` 数组：
```javascript
patentsData.push({
    id: 'P2026-XXX',
    title: "新专利标题",
    inventors: "完成人",
    number: "申请号",
    status: "authorized",
    type: "china",
    year: 2026
});
```

### 添加新项目
编辑 `modern-scripts.js` 中的 `projectsData` 对象

### 添加新系统平台
编辑 `modern-scripts.js` 中的 `systemsData` 数组
