## 💾 数据更新指南

### 添加新论文
编辑 `modern-scripts.js` 中的 `papersData.journal` 或 `papersData.conference`：

```javascript
papersData.journal.push({
    id: 'J2026-001',
    title: "论文标题",
    authors: "作者1, Yi Du*, 作者2",  // Yu Du* 会自动高亮
    journal: "期刊名称",
    year: 2026,
    sort_year: 2026,
    direction: "ai-for-science",  // ai-for-science, knowledge-graph, dataset
    type: "journal",
    meta: "SCI IF:XX, JCR Q1",
    url: "https://..."  // 可选
});
```

### 添加新专利
编辑 `modern-scripts.js` 中的 `patentsData`：

```javascript
patentsData.push({
    id: 'P2026-001',
    title: "专利标题",
    inventors: "完成人1, 杜一, 完成人2",
    number: "2026XXXXXXXX",
    status: "authorized",  // authorized, exam, applied
    type: "china",  // china, pct
    year: 2026
});
```

### 其他更新
- 科研项目：编辑 `projectsData`
- 系统平台：编辑 `systemsData`
- 标准、学生等：在渲染函数中直接修改数据
