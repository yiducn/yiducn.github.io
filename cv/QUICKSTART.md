## 🛠️ 快速更新数据

### 添加新论文（最常见操作）

编辑 `modern-scripts.js` 文件：

```javascript
// 找到 papersData 对象中的 journal 数组
papersData.journal.push({
    id: 'J2026-XXX',
    title: "你的论文标题",
    authors: "作者1, Yi Du*, 作者2",  // Yi Du* 会自动高亮为蓝色
    journal: "期刊名称",
    year: 2026,
    sort_year: 2026,
    direction: "ai-for-science",  // 选择: ai-for-science, knowledge-graph, dataset
    type: "journal",
    meta: "SCI IF:XX, JCR Q1 等信息",
    url: "https://论文链接.com"  // 可选
});
```

保存后，刷新页面即可看到新论文。

### 添加新专利

```javascript
patentsData.push({
    id: 'P2026-XXX',
    title: "专利标题",
    inventors: "完成人1, 杜一, 完成人2",
    number: "2026XXXXXXXXXXX",
    status: "authorized",  // 选择: authorized, exam, applied
    type: "china",  // 选择: china, pct
    year: 2026
});
```

### 其他类型数据

- **科研项目**：编辑 `projectsData.recent` 或 `projectsData.older`
- **大数据系统**：编辑 `systemsData`
- **标准、学生、兼职**：在 `renderStandards()` 等函数中修改

## 🎯 常见操作

### Q: 如何隐藏历史项目？
A: 点击"显示更多"按钮可以展开/收起历史项目

### Q: 如何搜索特定作者的论文？
A: 在搜索框输入作者名字（如"杜一"或"Yi Du"）

### Q: 能否支持多个条件组合查找？
A: 可以！所有条件自动组合。例如：选择"AI for Science" + "期刊论文" + "2024-2026"

### Q: 怎样添加新的研究方向分类？
A: 编辑 `modern-scripts.js` 中的 `getDirectionLabel()` 函数，并在论文数据中使用新分类

### Q: 如何将新页面设为默认首页？
A: 将 `index_zh_new.html` 重命名为 `index_zh.html`，备份原文件为 `index_zh_backup.html`
